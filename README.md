# XinApp
A simple browser that integrates some of my favorite website.

# Requirements
Xcode 7 and iOS 9 are required.

# Features
* 非WiFi网络环境下不加载图片(待优化)
* 使用3D Touch从主屏幕图标快速启动
* 使用Share Extension为应用添加数据
* 使用Background Fetch后台刷新数据
* 使用SFSafariViewController
